# Landing Page Project (Archived)

## Overview
This project is a simple **landing page** built using **HTML and CSS** as part of an early learning exercise. It was created while following **The Odin Project** curriculum to practice page layout, semantic HTML, and basic styling with CSS.

The page consists of common landing page sections such as a header, hero area, information cards, a quote section, a call-to-action, and a footer.

This project is now **archived** and is no longer actively maintained. It remains as a snapshot of early progress in learning front-end development fundamentals.

---

## Purpose
The main goals of this project were to:
- Practice writing **semantic HTML**
- Learn and apply **CSS layout techniques**
- Understand how to structure a complete web page
- Work with fonts, spacing, and visual hierarchy
- Follow a design specification and recreate it in code

This project was created purely for educational purposes.

---

## Features
- Header with logo text and navigation links
- Hero section with main heading, subtext, button, and image placeholder
- Information section with multiple content blocks
- Quote/testimonial section
- Call-to-action section with sign-up button
- Footer with copyright text
